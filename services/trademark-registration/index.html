Frequently Asked Questions

How long does trademark registration take?

18-24 months on average. But you get priority from application filing date, not approval date.

Can I use TM symbol before registration?

Yes, TM means "Trademark claimed". ¬Æ symbol can only be used AFTER registration is approved.

What are trademark classes?

45 classes categorizing goods (1-34) and services (35-45). Class 25 is clothing, Class 42 is IT services, etc. Choose relevant class(es).

Do I need separate trademark for logo and name?

Yes, recommended. Register word mark (brand name) and device mark (logo) separately for full protection.

What if someone objects to my trademark?

You get chance to respond. Trademark office decides. If rejected, can appeal to IPAB (Intellectual Property Appellate Board).

Is trademark registration valid internationally?

No, only in India. For international protection, file separately in each country or use Madrid Protocol for multiple countries.

Can I register personal name as trademark?

Yes, if not deceptively similar to existing marks. Many personal brands are registered trademarks (Ratan Tata, Amitabh Bachchan, etc.).
